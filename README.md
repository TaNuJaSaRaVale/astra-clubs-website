##  About the Project

The ASTRA Clubs Website is a modern, responsive, and elegant web application designed to represent the departmental AI & ML club.  
It serves as a central platform to showcase:

- Club vision and mission  
- Department and institute information  
- Events and initiatives  
- Team members  
- Contact and social media links  

The website emphasizes **clean UI**, **smooth navigation**, and **professional academic presence**.
